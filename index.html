Tipo de input Ejemplo Descripción/función
Semana

Permite al usuario seleccionar una semana y un año. Aparece un selector de fecha al ingresar al campo de entrada.
No es compatible con Firefox, Safari o Internet Explorer 11 (o versiones anteriores).
URL
Se usa para campos de entrada que deben contener una dirección URL. Dependiendo de la compatibilidad del navegador, el campo de URL se puede validar automáticamente cuando se envía. Algunos smartphones reconocen el tipo de URL y agregan ".com" al teclado para que coincida con la entrada de URL.
Tiempo
Permite al usuario seleccionar una hora (sin zona horaria): si el navegador lo admite, aparece un selector de hora al ingresar al campo de entrada. No es compatible con Safari o Internet Explorer 12 (o versiones anteriores).
Rango
Define un control para ingresar un número cuyo valor exacto no es importante (como un control deslizante). El rango predeterminado es de 0 a 100. Sin embargo, puede establecer restricciones sobre los números que se aceptan con los atributos min, max y step:

Mes
Permite al usuario seleccionar un mes y un año. No es compatible con Firefox, Safari o Internet Explorer 11 (o versiones anteriores).
Fecha local
Especifica un campo de entrada de fecha y hora, sin zona horaria. No es compatible con Firefox, Safari o Internet Explorer 12 (o anterior).
Color
Se utiliza para campos de entrada que deben contener un color. No es compatible con Internet Explorer 11 o Safari 9.1 (o versiones anteriores).
